### renewAccessToken
Renew Access Token



```javascript
// Promise
const promise =  
        merchant.renewAccessToken(
            { 
              body : value
            
         }
        );

// Async/Await
const data = await 
                    merchant.renewAccessToken(
                    { 
                       body : value
                    
                     });
```





| Argument  |  Type  | Required | Description |
| --------- | -----  | -------- | ----------- |
| body | [RefreshTokenRequest](#RefreshTokenRequest) | yes | Request body |


Use this API to renew access token

*Returned Response:*




[RefreshTokenResponse](#RefreshTokenResponse)
